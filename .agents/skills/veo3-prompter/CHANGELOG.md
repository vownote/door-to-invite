# Changelog

All notable changes to Veo 3.1 Prompter will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-12

### Added
- Initial release of Veo 3.1 Prompter skill
- Five-element prompt formula: Cinematography + Subject + Action + Context + Style/Audio
- Complete cinematography glossary with 50+ terms
- Audio direction support for dialogue, SFX, and ambient
- Timestamp prompting for multi-shot sequences (up to 8 seconds)
- 20+ categorized prompt examples across genres
- Advanced workflows documentation:
  - Reference images to video
  - First and last frame transitions
  - Image-to-video animation
  - Multi-prompt storyboard production
  - Iterative refinement with Gemini
- Style keywords for visual aesthetics, mood, and lighting
- Technical specs reference (duration, resolution, aspect ratio)

### Documentation
- Comprehensive README with quick start guide
- Three reference documents for progressive disclosure
- Source attribution to official Google documentation

## [Unreleased]
